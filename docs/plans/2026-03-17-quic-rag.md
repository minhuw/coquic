# QUIC RAG Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a local RFC-only QUIC knowledge base under `tools/rag/` that indexes `docs/rfc/`, stores embeddings in QdrantLocal, and exposes Codex-friendly lookup tools through FastMCP.

**Architecture:** A Python project in `tools/rag/` parses RFC text into canonical section records, extracts deterministic graph artifacts, embeds those records with a local model, writes them into a repo-local QdrantLocal state directory under `.rag/`, and serves query operations through a small FastMCP interface. The vector store is rebuildable generated state; source RFCs and deterministic code remain in Git.

**Tech Stack:** Python, `uv`, `pytest`, `qdrant-client` with `QdrantLocal`, `sentence-transformers`, `mcp` FastMCP SDK
